Graphical perception & aesthetics

Visualization size and effectiveness

Size is an important design heuristic for visualizations — after all, there's a balance between total information shown and how easy information is to locate. To better understand this relationship we looked at how data quantity, perceptual grouping, and the size of marks in visualizations affect how quickly and accurately users can search for information. We found that when categories encoded by color are grouped together, search is fast regardless of data quantity and mark size; yet if the data is not grouped, performance quickly degrades as data size increases or mark size decreases.

The relation between visualization size, grouping, and user performance. C.C. Gramazio, K.B. Schloss, D.H. Laidlaw. IEEE TVCG/InfoVis 2014.

Exploration of visualization design space

To classify existing visualizations, and to better explore visualization design space, we explored visual design patterns in hierarchal visualization techniques by clustering different techniques into a phylogenetic tree. We were able to trace visual relations between different rendering techniques and highlight several as-of-yet undiscovered hierarchal rendering visual styles. Finally, we developed an interactive exploration tool that allows users to weight 12 visualization design features to identify rendering techniques that align with the user's design needs.

Exploring hierarchical visualization designs using phylogenetic trees. S. Li, R.J. Crouser, G. Griffin, C.C. Gramazio, H. Schulz, H. Childs and R. Chang. Visualization and Data Analysis 2015.

Biology visualization

MAGI

We built MAGI to improve cancer researchers' ability to interactively explore their preliminary data alongside TCGA datasets — the world's largest public collection of cancer data. Not only does MAGI provide an easily importable, interactive visualization environment, but the tool also allows researchers to add public annotations to TCGA and other public datasets, thus annotating errors or underexamined areas in cancer data.

MAGI: Visualization and Collaborative Annotation of Genomic Aberrations. M.D.M. Leiserson, C.C. Gramazio, J. Hu, H. Wu, D.H. Laidlaw, B.J. Raphael. In Nature Methods.

Publisher

GD3

GD3 is a genomics visualization library meant for visualization novices. The library simplifies the construction of genomics visualizations by requiring less than ten lines of javascript code to create fully-interactive displays. GD3 currently supports six common, genomics-specific visualizations and we are constantly adding more.

Molli

Molli is a protein structure visualization tool. Aimed at improving protein rendering quality and speed, the tool uses coordinated multiple views for protein specialists to better compare structural properties across different proteins.

Molli: Interactive Visualization for Exploratory Protein Analysis. S.L. Su, C. Gramazio, D. Extrum-Fernandez, C. Crumm, L.J. Cowen, M. Menke, M. Strait. IEEE CG&A, 2012

Other research

Crowdsourcing evaluation platform

Professor Lane Harrison started the Experimentr project to provide researchers an easy way to launch complex crowdsourcing experiments. I've helped develop and steer the platform at various points in the development cycle. Built on node.js, Experimentr provides researchers with a stripped down, simple platform to launch Mechanical Turk surveys. Experimentr can support questionaires as well as more complex online experiments such as those that use response time.

Crowdsourcing methods

As part of a graduate human-computer interaction seminar lead by Professor Jeff Huang we researched the efficacy of different crowdsourcing techniques, while simultaneously generating a dataset of 2,200 faculty members in the Top-50 US computer science graduate programs. You can see the results here.

Crowdsourcing from Scratch: A Pragmatic Experiment in Data Collection by Novice Requesters. A. Papoutsaki, et al. AAAI HCOMP 2015.
